# Providers

Currently, promptctl supports the following providers:

## Google

Uses the Gemini API via `@google/generative-ai`.

**Configuration:**
Set `GOOGLE_API_KEY` environment variable.

Supported Models:
- `gemini-1.5-flash`
- `gemini-1.5-pro`
- `gemini-1.0-pro`
